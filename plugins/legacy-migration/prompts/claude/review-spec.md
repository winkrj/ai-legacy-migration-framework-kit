# Claude Spec Review Prompt

Review the supplied Discover, Specify and OpenSpec documents.

- unsupported assumptions
- missing requirements or scenarios
- evidence-level mistakes
- unresolved policy decisions
- non-goal and safety boundaries

Return required fixes, recommendations and Open Questions. Do not invent target behavior.
