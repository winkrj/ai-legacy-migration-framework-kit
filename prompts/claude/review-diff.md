# Claude Diff Review Prompt

Compare the supplied diff with approved OpenSpec requirements and Plan items.

Report:

- matched requirements
- out-of-scope changes
- missing validation
- safety or compatibility risks

Do not approve unrelated changes or claim execution not evidenced in the input.
